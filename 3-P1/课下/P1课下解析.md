## BlockChecker

难点在于，输入“end”后，理论上后续所有的输入都要被dismiss，  
可是跟着的如果是“a”，又矛盾了，  
所以trick在于，当end后面跟着字母时，要把想办法把res扳回来。  
比较烦的地方有两个：

1. 大小写转换，可以用case，也可以直接上function

2. 非阻塞赋值时，如果还要对其值进行比较，那么用的是其赋值前的值  

例子如下

    if(in == 1) begin
        cnt <= cnt + 1;
        if(cnt == 4) begin
            reset <= 1
        end
    end
